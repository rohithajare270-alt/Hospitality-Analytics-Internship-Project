Overview

This project was completed during a 3-month Data Analytics Internship focused on Hospitality Analytics.

The objective was to analyze hotel booking data, identify booking trends, monitor revenue performance, track occupancy metrics, and provide actionable business recommendations using data analytics tools.

The project involved complete data analysis lifecycle activities including data cleaning, transformation, querying, visualization, dashboard development, and business insight generation.

Internship Details
Internship Duration: 14 December 2025 – 16 March 2026
Domain: Hospitality Analytics
Role: Data Analyst Intern
Project Type: End-to-End Data Analytics Project
Business Problem

Hotels generate large volumes of booking data every day. However, without proper analysis it becomes difficult to:

Track revenue performance
Monitor occupancy rates
Identify booking patterns
Understand cancellation behavior
Optimize pricing and operational decisions

This project addresses these challenges through data-driven analysis and dashboard reporting.

Project Objectives
Analyze hospitality booking data
Identify revenue trends
Track hotel occupancy performance
Monitor booking status
Analyze customer booking behavior
Create interactive dashboards for decision making
Generate business recommendations based on insights
